# EJNMMI submission bundle

Everything needed to submit this paper to *European Journal of Nuclear Medicine and
Molecular Imaging* (EJNMMI), Original Article. Regenerated by `../../build_paper.sh`.

## What to upload (in EditorialManager)

| Order | File | Role |
|---|---|---|
| 1 | `manuscript_ejnmmi.docx` | Main manuscript (editable — most journals require DOCX) |
| 1b | `manuscript_ejnmmi.pdf` | Main manuscript (PDF, for reference / merged proof) |
| 2 | `cover_letter_ejnmmi.docx` / `.pdf` | Cover letter to the editor |
| 3 | `figures/Fig1.png` … `Fig6.png` | The six main figures, high resolution, one file each |
| 4 | `ESM_1_monte_carlo_validation.pdf` | Online Resource 1 (electronic supplementary material) |
| — | `submission_checklist_ejnmmi.md` | Author checklist — not uploaded; work through it before submitting |
| — | `submission_manifest.sha256` | Exact-hash manifest for the assembled package — not uploaded |

`ESM_1_monte_carlo_validation.png` is the raw supplementary figure, embedded in the ESM PDF.
The main manuscript PDF/DOCX contains figure legends only; main figures are uploaded separately
as Fig1-Fig6 to avoid caption/figure drift in the journal manuscript file.

Run `../../verify_reproducibility.sh` before upload. Set `STRICT_BINARY=1` to verify the exact
submission files against `submission_manifest.sha256`.

## Notes

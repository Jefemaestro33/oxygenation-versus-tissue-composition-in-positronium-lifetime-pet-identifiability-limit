EJNMMI LaTeX flat upload package

Upload order / file type suggestions in Editorial Manager:
1. main.tex                         -> Manuscript (first item)
2. sn-jnl.cls                       -> LaTeX Supporting File(s)
3. Fig1.png ... Fig6.png            -> Figure
4. manuscript_latex_compiled.pdf    -> Supplementary material / PDF reference, if the system asks for a PDF reference
5. ESM_1_monte_carlo_validation.pdf -> Electronic Supplementary Material / Online Resource
6. cover_letter_ejnmmi.pdf          -> Cover Letter step/type, not Manuscript

All files are intentionally in one folder level. Do not upload this README unless you want an internal checklist.
References are embedded in main.tex using thebibliography, so no .bib/.bst file is required.
